# St. Paul Crime Map - Single Page Application

## Setup and Running Instructions

1. Clone the repository

2. Install dependencies:
   ```
   npm install
   ```

3. Build the application:
   ```
   npm run build
   ```

4. Start the server:
   ```
   node rest_server.mjs
   ```

5. Visit `http://localhost:8000` in your browser

The server runs on port 8000 and serves both the API endpoints and the web application.
